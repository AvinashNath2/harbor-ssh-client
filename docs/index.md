# HarborSCP Documentation

Welcome to the HarborSCP documentation. HarborSCP is a native SSH/SFTP desktop client for developers, built with Tauri and Rust.

## Pages

- [Installation](installation.md) — Download a release or build from source
- [User Guide](user-guide.md) — Connecting, browsing files, terminal, transfers, tunnels, session log
- [Configuration](configuration.md) — Connection profiles, SSH keys, keyboard shortcuts
- [FAQ](faq.md) — Common questions and troubleshooting
- [Architecture](architecture.md) — For contributors: internals, session model, IPC pattern
